# MemoriaV2  

> 一款专注于视频缓存管理与混流导出的桌面端工具软件，支持从本地缓存文件（如B站等平台）读取视频数据，并提供预览、混流、导出等功能。

---

## 概述

Memoria是一款面向视频缓存文件管理的桌面应用程序，旨在支持用户：  
- **读取本地视频缓存文件**(含本地已缓存下架视频)
- **预览视频内容**（双屏播放对比、网页搜索、下架检测）
- **混流导出**（将分离的音视频文件合并为常见通用格式）
- **可视化视频元数据**（标题、UP主、系列、弹幕等信息）

### 关于数据读取
支持用户从不同来源导入视频缓存数据

**即将支持的数据源：**
| 导入方式 | 说明 |
|---------|------|
| 本设备导入 | 从本地文件系统读取缓存文件 |
| WLAN连接导入 | 通过无线局域网连接设备传输 |
| 设备数据线导入 | 通过USB数据线连接设备传输 |

### 核心特性

- **多平台缓存支持**：将重点支持B站/油管缓存结构
- **零配置要求**：内置FFmpeg环境
- **批量处理能力**：支持多文件同时导入与混流
- **实时预览**：无需导出即可预览缓存视频内容
- **智能检测**：提供网页搜索对比，自动识别视频是否下架
- **双播放预览**：可双播放器对比播放，查看缓存与平台视频差别
- **多格式兼容**：支持主流视频格式导出
